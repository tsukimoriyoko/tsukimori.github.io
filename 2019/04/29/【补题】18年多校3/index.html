Euler Function

Description

Giving an integer $k$, your task is to find the $k-th$ smallest positive integer $n$, that $\phi(n)$ is a composite number.

Solution

由$gcd(n,\ k)=gcd(n,\ n-k)$，易得$n>2$时$\phi(n)$为偶数，则不难证明当$n\le 7$时，$\phi(n)$恒为合数。
